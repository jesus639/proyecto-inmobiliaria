# proyecto-inmobiliaria
Descripcion del proyecto

Este proyecto es una aplicacion creada en Java utilizando NetBeans y Swing. Su proposito es gestionar la informacion de una inmobiliaria por medio de varios formularios que permiten registrar, consultar y administrar usuarios, propiedades, visitas, ventas, arriendos y pagos. Toda la informacion se almacena en archivos de texto, lo que permite guardar los datos sin necesidad de una base de datos avanzada.

La aplicacion esta dividida en diferentes modulos, cada uno representado por un formulario.

1. Gestion de usuarios

El sistema permite registrar y administrar distintos tipos de usuarios, como agentes, propietarios y clientes. Cada tipo de usuario tiene su propio formulario de entrada. Los datos se guardan en un archivo llamado usuarios.txt, y se pueden consultar o gestionar desde un formulario general dedicado a usuarios.

2. Gestion de propiedades

El proyecto incluye formularios para registrar propiedades, ver su estado y consultar informacion relacionada con casas disponibles. Cada propiedad puede tener imagenes asociadas y detalles como su estado, tipo y datos del propietario. Toda la informacion se almacena en el archivo propiedades.txt.

3. Gestion de visitas

El sistema permite programar visitas a las propiedades, registrar clientes interesados y visualizar las visitas programadas o realizadas. Estas visitas se guardan en visitas.txt y pueden consultarse desde formularios especificos.

4. Gestion de ventas y arriendos

El proyecto ofrece formularios para revisar ventas realizadas y propiedades arrendadas. La aplicacion guarda la informacion de estas operaciones en archivos separados, los cuales son mostrados en las ventanas correspondientes.

5. Gestion de pagos

Existe un modulo para consultar pagos registrados dentro del sistema. Estos datos se guardan en pagosregistros.txt y pueden visualizarse desde el formulario de pagos.

6. Sesion y roles

El sistema utiliza clases para manejar el inicio de sesion y el tipo de usuario. Esto permite controlar que informacion puede ver o modificar cada rol, organizando mejor el funcionamiento de toda la aplicacion.

Proposito del proyecto

Este proyecto fue desarrollado con la finalidad de practicar conceptos fundamentales en Java Swing, manejo de archivos de texto, programacion orientada a objetos y administracion de datos entre multiples formularios. Su estructura representa el funcionamiento basico de un sistema inmobiliario sencillo, permitiendo experimentar con formularios, roles de usuario y gestion de informacion sin necesidad de una base de datos compleja.
